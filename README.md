# 🏰 행운 타워 디펜스

아빠와 아들이 함께 만드는 운빨 타워디펜스 게임! (운빨존망겜 스타일)

## 🎮 게임 방법

- **🎲 소환** 버튼: 골드를 써서 랜덤 캐릭터를 뽑아요 (일반 55% / 희귀 30% / 영웅 12% / 전설 3%)
- **합성**: 같은 캐릭터 둘을 차례로 누르면 한 등급 위의 랜덤 캐릭터가 돼요. 5% 확률로 **2단계 점프 대박!** 🎉
- **이동/교체**: 캐릭터를 누른 뒤 빈 칸을 누르면 이동, 다른 캐릭터를 누르면 자리 교체 (무료지만 **1초간 공격 불가** ⏳ — 노란 원이 줄어드는 동안 못 싸워요)
- **패배 조건**: 트랙을 도는 적이 **60마리** 쌓이면 게임 오버
- 5웨이브마다 **보스** 👹 등장!

## ▶️ 실행 방법

- **컴퓨터에서**: `index.html` 파일을 더블클릭하면 브라우저에서 바로 실행돼요.
- **휴대폰에서 (같은 와이파이)**: 컴퓨터에서 아래 명령 실행 후, 폰 브라우저로 `http://컴퓨터IP:3456` 접속
  ```
  npx serve -l 3456 .
  ```
- **친구들과 (인터넷 공개)**: GitHub Pages에 올리면 링크만 보내면 누구나 플레이 가능 — 아래 배포 방법 참고

💡 검사·쥬라사우르스·나무꾼·캣머신은 사거리가 짧아서 **트랙에 가까운 바깥쪽 칸**에 놓아야 해요. 이동 기능으로 옮겨보세요!

## ⚙️ 밸런스 조정 (아들과 함께 실험해보세요!)

`index.html` 상단의 상수들을 바꾸면 게임이 달라져요:

| 상수 | 의미 | 현재값 |
|---|---|---|
| `ENEMY_LIMIT` | 몇 마리 쌓이면 패배 | 60 |
| `ROUND_LEN` | 웨이브 간격(초) | 20 |
| `SUMMON_BASE` | 소환 시작 비용 | 20 |
| `JACKPOT_CHANCE` | 합성 대박 확률 | 0.05 (5%) |
| `GRADE_PROB` | 등급별 소환 확률 | 55/30/12/3% |
| `UNITS` 배열 | 캐릭터 공격력/속도/사거리 | - |

## 🚀 GitHub Pages 배포 (친구들에게 공유)

1. github.com 가입 → 새 저장소(repository) 만들기 (예: `tower-defense`, Public)
2. 이 폴더 파일들을 업로드
3. 저장소 Settings → Pages → Branch를 `main`으로 선택 → Save
4. 몇 분 뒤 `https://아이디.github.io/tower-defense` 주소가 생겨요 → 친구들에게 링크 전송!
