# 📔 타워 디펜스 777 개발 일지

> 구예준 STUDIO — 아빠와 아들이 함께 만든 첫 게임의 기록
>
> 🎮 플레이: https://kongpot9.github.io/tower-defense/

## 프로젝트 소개

운빨존망겜(Lucky Defense) 스타일의 모바일 웹 타워디펜스.
캐릭터 그림과 게임 기획은 **구예준**(아들), 개발은 아빠와 Claude가 맡았다.
설치 없이 폰 브라우저로 바로 플레이할 수 있고, 친구들과 온라인 랭킹으로 경쟁한다.

## 개발 연대기

### 2026-07-17 — 탄생

- **v1 출시**: 랜덤 소환 / 합성(5% 대박) / 이동·교체 / 웨이브 방어의 핵심 루프 완성
- 아들이 그린 **캐릭터 12종** 전원 탑재 (일반 검사·궁수·농부 / 희귀 쥬라사우르스·원까태인·탱크보이 / 영웅 데스삐에로·나무꾼·다크라이더 / 전설 레드드래곤·도사·캣머신)
- 캐릭터마다 다른 공격 이펙트: 칼베기, 화살, 낫, 물기, 야구공+불꽃공, 포탄, 펀치, 도끼, 수리검, 화염 브레스, 부채, 미사일
- 게임 이름을 **"타워 디펜스 777"** 로 확정, 화면 상단에 구예준 STUDIO 크레딧
- **신화 등급 신설** (게임 역사상 드물게 신화 < 전설 — "신화는 어디까지나 이야기지만 전설은 실제로 있었다"는 구예준 세계관)
- 신화 4종 추가: 능력자광부(폭탄 설치), 떡장수(보스 무시·4연발 송편), 낭인(발도술), 천사 도베르만(8% 즉사)
- GitHub Pages **정식 배포** — 링크만 있으면 누구나 플레이
- 일시정지 버튼 (플레이어 피드백 1호)

### 2026-07-18 — 대확장

- **밸런스 패치**: 궁수 버프("하층민으로 격상" — 개발자 공식 코멘트), 원까태인·쥬라사우르스·데스삐에로 버프, 레드드래곤 너프
- **모드 분리**: 🎮 일반 모드(끝없는 웨이브) / 🗺️ 레전드 스토리(스테이지 캠페인)
- **레전드 스토리 3개 장, 총 42스테이지** (전체 스테이지 기획: 구예준)
  - 1장 14개: 초원 ~ 마왕성 (물먹은 섬, 아주 슬픈 추석, 빅뱅, 우그하문, 카타스트로프…)
  - 2장 12개: 아직 끝나지 않은 초원 ~ 마왕성 (적색의 식탁보, 조롱거리, 재질좋은 우그하문…)
  - 3장 16개: 이제야 끝이 보이는 초원 ~ **바가지 요금에 흐느끼는 밤** (팸 팸 너 고기 스팸, 가울리즘, 드라이트론 우그하문, 아버지의 낚시터…)
  - 갈수록 적 체급 상승 (체력 최대 3배, 보스 배율 51배), 기믹 3종: 몬스터 러시 / 혹한 / 쌍둥이 보스
- **닉네임 + 온라인 랭킹 오픈** (Firebase): 일반 모드 최고 웨이브 / 스토리 ⭐ 클리어 수
